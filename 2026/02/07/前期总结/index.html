Linux

目录路径 描述
/bin Binary的缩写，存放最经常使用的命令的可执行文件，本质是/usr/bin的软链接。
/sbin s代表Super User，存放系统管理员使用的系统管理程序，本质是/usr/sbin的软链接。
/home 存放普通用户的主目录，每个用户都有一个以自己账号命名的目录。
/root 系统管理员（超级权限者）的用户主目录。
/lib 系统开机所需最基本的动态连接共享库，类似于Windows的DLL文件，几乎所有应用程序都需要。
/etc 存放所有系统管理所需要的环境配置文件和子目录。
/usr 非常重要的目录，存放用户应用程序和文件，类似于Windows下的Program Files目录。
/boot 存放启动Linux时使用的核心文件（包括连接文件和映射文件），请勿将自己的文件放在这里。
/run 存放系统运行产生和需要的一些重要文件。
/proc 系统中运行的进程的虚拟映射目录，可通过访问此目录获取系统信息。
/srv Service缩写，用于存放系统运行时数据（特别是服务产生的数据），默认为空。
/sys Linux 2.6内核的新特性，该目录下安装了2.6内核中新出现的一个文件系统。
/tmp 存放临时文件，过一段时间会自动清除。
/dev 类似于Windows的设备管理器，以文件形式存储所有硬件。
/media Linux自动识别设备（如音响、耳机）后，会将识别的设备挂载到此目录下。
/mnt 供用户临时挂载其他文件系统（如U盘、移动硬盘）的目录，默认为空。
/opt 主要用于安装可选的或第三方应用程序（如MySQL），默认为空。
/var 存放不断扩充的内容，通常将经常修改的目录（如各种日志文件）放在此目录下。

MySQL

数据库操作

操作说明 SQL 语句
创建数据库 create database if not exists MyDataBase;
查看所有数据库 show databases;
查看数据库的创建语句 show create database MyDataBase;
切换数据库 use MyDataBase;
修改数据库的字符集为Latin1（不支持中文） alter database MyDataBase character set Latin1 collate Latin1_general_ci;
删除数据库 drop database if exists MyDataBase;

表操作

操作说明 SQL 语句
创建数据库 create database if not exists MyDataBase;
切换数据库 use MyDataBase;
查看数据库中所有表 show tables;
查看表结构 desc teacher;
查看建表语句 show create table teacher;
添加字段到表末尾 alter table teacher add address varchar(200);
添加字段到表开头 alter table teacher add age int first;
添加字段到指定位置（age 之后） alter table teacher add heart_rate float after age;
删除字段 alter table teacher drop address;
修改字段数据类型 alter table teacher modify heart_rate double;
修改字段名 alter table teacher change heart_rate bpm double;
移动字段位置到表开头 alter table teacher modify bpm double first;
移动字段位置到另一个字段之后 alter table teacher modify bpm double after age;
修改表名 alter table teacher rename teacher_info;
删除表 drop table teacher_info;

增删改查

操作说明 SQL 语句示例
插入一条数据（按字段顺序） insert into teacher values(1, '张三', '男', '1990-01-01', 10000, '12345678901');
插入一条数据（指定字段） insert into teacher (id, name, gender, birthday, salary, tel) values(2, '李四', '男', '1992-05-12', 12000, '13800001111');
查询所有数据 select * from teacher;
查询指定字段 select name, salary from teacher;
带条件查询（WHERE） select * from teacher where gender = '男';
查询并排序 select * from teacher order by salary desc;
分组查询（GROUP BY） select gender, avg(salary) from teacher group by gender;
更新（修改）数据 update teacher set salary = 15000 where name = '张三';
删除所有数据（保留表结构） delete from teacher;
删除满足条件的数据 delete from teacher where name = '郑十';
删除姓名为“郑十”的记录 delete from teacher where name = '郑十';
删除整张表（DDL，非DML） drop table if exists teacher;
清空表（DDL，效率高于DELETE） truncate table teacher;
修改满足条件的行 update 表名 set 字段名 = 值, 字段名 = 值 where 条件;

运算

运算符类型 操作说明 SQL 语句示例
算术运算符 加法 / 减法 / 乘法 / 除法 / 取余 select salary + 500, salary - 200, salary * 1.1, salary / 12, salary % 3 from t_employee;
比较运算符 等于 select * from t_employee where name = '张三';
不等于（两种写法） select * from t_employee where salary != 15000;
select * from t_employee where salary <> 15000;
大于 / 小于 / 大于等于 / 小于等于 select * from t_employee where salary > 15000;
select * from t_employee where age <= 30;
逻辑运算符 AND（且） select * from t_employee where salary > 15000 and gender = '男';
OR（或） select * from t_employee where salary > 15000 or did = 1;
NOT（非） select * from t_employee where not (salary between 15000 and 20000);
XOR（异或：两者有且仅有一个为真） select * from t_employee where salary > 15000 xor did = 1;
select * from t_employee where (salary > 15000) ^ (did = 1);
范围运算符 BETWEEN … AND（包含边界） select * from t_employee where salary between 15000 and 20000;
NOT BETWEEN … AND select * from t_employee where salary not between 15000 and 20000;
集合运算符 IN（匹配列表中的任意值） select * from t_employee where did in (1, 3, 5);
NOT IN select * from t_employee where did not in (2, 4);
模糊匹配运算符 LIKE（通配符 % 任意多个字符，_ 单个字符） select * from t_employee where name like '张%'; – 姓名以“张”开头
select * from t_employee where name like '_三'; – 两个字符，第二个是“三”
空值判断 IS NULL select * from t_employee where bonus is null;
IS NOT NULL select * from t_employee where bonus is not null;
位运算符 按位与 / 或 / 异或 / 取反 select 5 & 3, 5 | 3, 5 ^ 3, ~1;
模糊匹配 查询带有love select * from t_employee where ename like '%love%';
以 你 开头 select * from t_employee where ename like '你%';

例如：conda remove matplotlib
